# 🌐 Personal Portfolio Website

A responsive and modern personal portfolio website built using **HTML** and **CSS**. This project showcases my skills, projects, and professional background in a clean and visually appealing layout.

---

## 🚀 Features

* **Header Section**
  Displays my name along with a brief introduction.

* **About Me**
  Includes my profile image and a short bio highlighting my background, skills, and experience.

* **Skills Section**
  Showcases my technical skills such as HTML, CSS, JavaScript, MySQL, and more.

* **Projects Section**
  Displays my work with project titles, descriptions, and preview images.

* **Resume Download**
  Option to download my resume in PDF format.

* **Contact Section**
  Contains my contact details including email and phone number.

* **Footer**
  Includes copyright information.
